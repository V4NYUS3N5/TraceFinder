# WebTrail 模块功能说明

## 整体架构

```
main.py                    ← CLI入口，协调各模块
├── extractors/
│   ├── chromium.py        ← Chromium内核浏览器数据提取
│   └── firefox.py         ← Firefox浏览器数据提取
├── system.py              ← 系统级痕迹提取
├── reporter.py            ← 报告生成 + 可疑标记 + 统计摘要
└── utils.py               ← 公共工具函数
```

---

## main.py — 主程序入口

- 命令行参数解析（`--output` 输出文件、`--json` JSON导出、`--quiet` 静默模式）
- 依次调用 Chromium 提取器 → Firefox 提取器 → 系统提取器，汇总全部痕迹
- 调用 `generate_report()` 生成报告并输出到控制台或文件
- 输出末尾一行简要统计（总计XX条，可疑XX条）

---

## extractors/chromium.py — Chromium 内核浏览器提取器

支持 **Chrome / Edge / Brave / Opera / 360浏览器** 共5款，每款提取7个维度：

| 维度 | 数据来源 | 方法 |
|------|----------|------|
| **浏览历史** | `Profile/History` SQLite DB → `urls` 表 | 取最近200条，含URL、标题、最后访问时间、访问次数 |
| **书签** | `Profile/Bookmarks` JSON文件 | 递归遍历书签栏/其他书签所有节点 |
| **下载记录** | `Profile/History` → `downloads` 表 | 文件名、大小(MB)、时间、状态(已完成/已取消/中断) |
| **Cookie域名** | `Profile/Network/Cookies` 或 `Profile/Cookies` | 按域名聚合统计Cookie数量，取Top50 |
| **登录凭据** | `Profile/Login Data` → `logins` 表 | 按来源URL统计凭据总量（不输出明文密码） |
| **会话标签页** | `Profile/Sessions/` 目录 | 文件名、大小、最后修改时间 |
| **扩展插件** | `Profile/Extensions/` 目录 | 读取 `manifest.json` 获取扩展名 + 安装时间 |

多Profile支持：自动读取 `Local State` JSON 中的 `profiles_order` 列表。

---

## extractors/firefox.py — Firefox 提取器

遍历 `%APPDATA%/Mozilla/Firefox/Profiles/` 下所有Profile，每个提取6个维度：

| 维度 | 数据来源 | 说明 |
|------|----------|------|
| **浏览历史** | `places.sqlite` → `moz_places` 表 | 最近200条，PRTime微秒时间戳 |
| **书签** | `places.sqlite` → `moz_bookmarks` + `moz_places` 联查 | 最近100条 |
| **下载记录** | `places.sqlite` → `moz_annos`（`downloads/destinationFileURI` annotation） | 最近50条 |
| **Cookie域名** | `cookies.sqlite` → `moz_cookies` 表 | 按域名聚合，取Top50 |
| **登录凭据** | `logins.json` | 按hostname统计，不输出明文 |
| **扩展插件** | `extensions.json` | 仅输出已启用的扩展名 |

## system.py — 系统级浏览器痕迹

| 维度 | 数据来源 | 方法 |
|------|----------|------|
| **浏览器启动记录** | 注册表 `HKCU\...\UserAssist\{CEBFF5CD-...}\Count` | ROT13解码后过滤 chrome/edge/firefox 等进程名，解析64字节偏移处FILETIME |
| **DNS缓存** | `ipconfig /displaydns` 命令输出 | 过滤CDN/内部域名，去重输出最近访问域名 |
| **Prefetch** | `C:\Windows\Prefetch\*.pf` 文件 | 匹配浏览器进程名的预读取文件及其最后修改时间 |

---

## reporter.py — 报告生成器

- **时间线排序**：按时间倒序排列所有有精确时间戳的痕迹
- **可疑标记**：基于关键词列表（密码/token/exploit/hack/破解/漏洞/勒索/HackBar 等）和可疑域名（pastebin/mega.nz 等）自动标记
- **报告格式**：

  ```
  头部 → 时间线主体（含可疑 !!可疑!! 标签）→ 统计信息 → 统计摘要
  ```

- **统计摘要**包含：总痕迹数、按类型分布、按来源分布、可疑标记详情

## utils.py — 公共工具

| 函数 | 功能 |
|------|------|
| `filetime_to_datetime()` | Windows FILETIME (8字节小端) → Python datetime |
| `chrome_time_to_dt()` | Chromium时间戳（1601年起微秒数）→ datetime |
| `firefox_time_to_dt()` | Firefox PRTime（1970年起微秒数）→ datetime |
| `read_sqlite_copy()` | 复制SQLite DB到临时文件避免浏览器锁定冲突，返回连接和临时文件路径 |
